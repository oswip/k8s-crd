# CRD Backward Compatibility

## Makefile
To shorten kubectl commands, there's a [Makefile](Makefile)
- To wipe the CRD and CRs
```
make clean
```

- To install a CRD
```
make crd v=1
```
- To create a CR
```
make cr v=1
```
